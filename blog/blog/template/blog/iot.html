What Is Internet Of Things ?

IOT is the latest upgrowing technology. which help us to connect with the real world.

devices based on it are helping to communicate and check various physical activities. which were not possible in everyday life.

the main thing which make iot popular is that. here the devices are connected to internet which make them able to send and receive data

Ex:-counting pulse rate, checking number of steps. etc






Real Life Examples Of IOT

1. Steps Counter : an iot product used to know the number of steps walked

2. Pulse Rate Counter : an iot product used to count pulserate

3. Tracker : a tracker is used to trace the live or last location

4. Sensor : sensor is used to detect the motion and respond to it
